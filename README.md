# Kairozun — Roblox Glass Overlay

Мощный стеклянный overlay для Roblox с записью экрана, фоторедактором, 100+ игровыми фильтрами, мониторингом системы, поиском игроков и многим другим.

Powerful glass overlay for Roblox with screen recording, photo editor, 100+ game filters, system monitoring, player lookup, and more.

## Features / Возможности

- 🔲 **Glassmorphism UI** — полностью стеклянный дизайн без рамок
- 📊 **FPS & Ping** — мониторинг в реальном времени с цветовой индикацией
- 🎬 **Screen Recording** — запись экрана с 5 пресетами качества (Ultra/Max/High/Medium/Low), H.264
- 📸 **Screenshots & Editor** — скриншоты, галерея, редактор с фильтрами, текстом, стикерами, обрезкой
- 🖼️ **Collage Creator** — создание коллажей из 2–9 скриншотов
- 🎨 **100+ Game Filters** — реалтайм цветовые фильтры (Noir, Vintage, Neon, Cyber и др.)
- 👤 **Account Card** — аватар, никнейм, друзья, подписчики, возраст аккаунта
- 👥 **Friends & Players** — список друзей онлайн, игроки на сервере с аватарами
- 🔍 **Player Lookup** — поиск любого игрока по нику с профилем, группами и бейджами
- 💻 **System Monitor** — CPU, GPU, RAM, диски, сеть, температура, ОС
- 📜 **Game History** — автоматическая запись сессий с временем, регионом, длительностью
- 🌐 **Server Details** — IP сервера, регион, пинг, FPS сервера, Place/Universe ID
- 💀 **Kill Feed** — лента убийств для Battle Royale игр
- 🎭 **Full Customization** — масштаб виджетов, прозрачность, 9 акцентных цветов
- 🌐 **RU / EN** — полная двуязычная локализация
- ⚡ **Always on Top** — всегда поверх игры, прозрачный для кликов

## Hotkeys / Горячие клавиши

| Hotkey | Action | Rebindable |
|--------|--------|-----------|
| `Alt+0` | Open/Close Settings | ✓ |
| `F9` | Take Screenshot | ✓ |
| `F10` | Start/Stop Recording | ✓ |
| `Alt+F4` | Quit Application | ✓ |

## Install / Установка

Скачайте установщик из [Releases](../../releases).

## Dev / Разработка

```bash
npm install
npm start
```

## Build / Сборка

```bash
npm run build
```

Установщик будет в папке `dist/`.
