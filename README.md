# Kairozun — Roblox Glass Overlay

Стеклянный overlay для Roblox с мониторингом FPS, пинга, нагрузки системы и килфидом.

Glass overlay for Roblox with FPS monitoring, ping, system load, and kill feed.

![Electron](https://img.shields.io/badge/Electron-41-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-green)

## Features / Возможности

- 🔲 **Glassmorphism UI** — полностью стеклянный дизайн без рамок
- 📊 **FPS & Ping** — мониторинг в реальном времени (верхний левый угол)
- 💀 **Kill Feed** — лента убийств (верхний правый угол)
- 💻 **CPU & RAM** — нагрузка системы (нижний левый угол)
- 🌐 **RU / EN** — двуязычный интерфейс
- ⚡ **Always on top** — всегда поверх игры, прозрачный для кликов

## Hotkeys / Горячие клавиши

| Hotkey | Action |
|--------|--------|
| `Alt+8` | Open/Close Settings / Открыть настройки |
| `Ctrl+Shift+H` | Show/Hide Overlay / Скрыть оверлей |

## Install / Установка

Скачайте установщик из [Releases](../../releases).

## Dev / Разработка

```bash
npm install
npm start
```

## Build / Сборка

```bash
npm run build
```

Установщик будет в папке `dist/`.
