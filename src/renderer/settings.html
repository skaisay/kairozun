Compliance & Safety
Kairozun is a safe external overlay that does not violate Roblox Terms of Use. The application does not inject code into Roblox, does not modify game memory, does not automate gameplay, and does not provide unfair advantages. It only reads publicly available log files and uses official Roblox public APIs.
